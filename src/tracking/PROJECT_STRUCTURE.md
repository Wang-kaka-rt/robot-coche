# ROS2 Object Tracking Package - 项目结构

## 完整目录结构

```
tracking/
│
├── package.xml                    # ROS2 包元数据（依赖、版本等）
├── setup.py                       # Python 包安装配置
├── setup.cfg                      # 构建和测试配置
├── .gitignore                     # Git 忽略文件配置
│
├── README.md                      # 包功能说明和使用指南
├── INSTALL.md                     # 详细安装和构建说明
├── EXAMPLES.md                    # 丰富的使用示例
├── params.yaml                    # 推荐参数配置文件
│
├── launch/
│   └── object_tracking.launch.py  # ROS2 启动文件
│
├── resource/
│   └── tracking                   # ROS2 资源文件
│
├── test_local/
│   └── test_tracker.py            # 本地测试脚本（无需硬件）
│
└── tracking/                      # Python 包源代码
    ├── __init__.py                # 包初始化
    ├── object_tracker_node.py     # 主跟踪节点（PID 控制）
    ├── pid_controller_node.py     # 独立 PID 控制器节点
    └── tracking_utils.py          # 工具函数和类
```

## 核心文件说明

### 1. 配置文件

#### [`package.xml`](package.xml)
- **作用**: ROS2 包的元数据文件
- **内容**: 包名、版本、依赖、作者信息
- **重要性**: ⭐⭐⭐⭐⭐ (必需)

#### [`setup.py`](setup.py)
- **作用**: Python 包的安装配置
- **内容**: 包名、版本、依赖、入口点
- **重要性**: ⭐⭐⭐⭐⭐ (必需)

#### [`setup.cfg`](setup.cfg)
- **作用**: 构建和测试配置
- **内容**: pytest 配置、开发依赖
- **重要性**: ⭐⭐⭐

#### [`params.yaml`](params.yaml)
- **作用**: 推荐参数配置集合
- **内容**: 6 种场景的参数预设
- **重要性**: ⭐⭐⭐⭐

### 3. 源代码文件

#### [`tracking/object_tracker_node.py`](tracking/object_tracker_node.py)
- **作用**: 主跟踪节点（核心逻辑）
- **功能**:
  - PID 控制器实现
  - 状态机管理（LOST, SEARCHING, TRACKING, FOLLOWING）
  - 障碍物避免
  - 电池监控
  - 自适应速度控制
- **代码行数**: ~450 行
- **重要性**: ⭐⭐⭐⭐⭐

#### [`tracking/pid_controller_node.py`](tracking/pid_controller_node.py)
- **作用**: 独立 PID 控制器节点
- **功能**:
  - 通用 PID 控制服务
  - 可被其他节点复用
  - 参数可配置
- **代码行数**: ~120 行
- **重要性**: ⭐⭐⭐

#### [`tracking/tracking_utils.py`](tracking/tracking_utils.py)
- **作用**: 工具函数和类
- **功能**:
  - 距离估算函数
  - 焦距计算函数
  - 指数移动平均滤波器 (EMA)
  - 一维卡尔曼滤波器
  - 运动剖面生成器
  - 角度归一化
  - 转向角计算
- **代码行数**: ~250 行
- **重要性**: ⭐⭐⭐⭐

### 4. 启动和测试文件

#### [`launch/object_tracking.launch.py`](launch/object_tracking.launch.py)
- **作用**: ROS2 启动配置
- **功能**:
  - 启动 object_tracker 节点
  - 配置默认参数
  - 支持命令行参数覆盖
- **重要性**: ⭐⭐⭐⭐⭐

#### [`test_local/test_tracker.py`](test_local/test_tracker.py)
- **作用**: 本地测试脚本
- **功能**:
  - 模拟物体检测数据
  - 测试跟踪逻辑
  - 无需硬件即可测试
  - 5 阶段测试序列
- **代码行数**: ~130 行
- **重要性**: ⭐⭐⭐⭐

```
┌──────────────┐

       │ 检测到物体                           │
       ▼                                      │

       │ 重新检测到物体                       │
       ▼                                      │
┌──────────────┐                              │
│  TRACKING    │                              │
│   (跟踪)     │                              │
│              │                              │
```

## PID 控制框图

```
                    ┌─────────────┐
    目标位置 ──────►│  误差计算   │
    (图像中心)      │             │
                    │ error =     │
                    │ target -    │
                    │ current     │
                    └──────┬──────┘
                           │
                           ▼
                    ┌─────────────┐
                    │  PID 控制器  │
                    │             │
                    │ P: Kp*error │
                    │ I: Ki*∫error│
                    │ D: Kd*d/dt  │
                    │             │
                    │ output =    │
                    │ P + I + D   │
                    └──────┬──────┘
                           │
                           ▼
                    ┌─────────────┐
    速度命令 ──────►│  速度限制   │
    (/cmd_vel)     │  和滤波     │
                   └─────────────┘
```

## 依赖关系

### ROS2 依赖
- `rclpy`: ROS2 Python 客户端库
- `geometry_msgs`: 几何消息类型（Point, Twist）
- `sensor_msgs`: 传感器消息类型（Range, BatteryState）
- `std_msgs`: 标准消息类型（Bool, Float32MultiArray, Int32）

### Python 依赖
- `numpy`: 数值计算
- `time`: 时间处理
- `threading`: 多线程支持
- `enum`: 枚举类型
- `typing`: 类型提示

### 系统依赖
- OpenCV (cv_bridge): 图像处理（视觉节点使用）
- ROS2 Humble 或更高版本

## 关键特性

### 1. PID 控制
- 独立的角速度和线速度 PID 控制器
- 抗积分饱和
- 输出限幅
- 状态重置机制

### 2. 状态机
- 4 个状态：LOST, SEARCHING, TRACKING, FOLLOWING
- 平滑状态转换
- 超时处理
- 障碍物优先

### 3. 安全特性
- 障碍物检测和规避
- 电池监控和低电量保护
- 速度限制
- 紧急停止

### 4. 可配置性
- 20+ 可调参数
- 支持运行时参数调整
- 预设场景配置
- 启动文件参数化

## 使用场景

### 适用场景
✅ 物体跟随（球、人、特定目标）
✅ 自主导航
✅ 避障行驶
✅ 教育演示
✅ 原型开发

### 不适用场景
❌ 高速跟踪（>1 m/s）
❌ 多目标跟踪
❌ 3D 定位
❌ SLAM 建图
